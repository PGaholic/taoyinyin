淘印印如何做到零元打印？

广告商需要在学校投放广告，宣传自己同学们需要打印资料，印资料需要花钱所以我们希望通过这样一个模式，把这两者结合既能满足广告商们的打响自己知名度的需求也能帮助大家直接不花钱得打印资料
免费的代价，就是存在广告，但同样作为学生，我们更能体会大家的感受所以把广告放在不影响大家阅读的地方。
